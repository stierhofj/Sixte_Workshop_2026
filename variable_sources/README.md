This directory contains multiple SIXTE simulation pipelines, showing how to generate SIMPUT files, run simulations, and analyze their output, focused on variable sources.

In the `basic_sims` example, you have already seen how to define a non-periodic light curve. The exercises here show other ways to define variability.

# periodic_lc
Build a SIMPUT using an ASCII light curve containing a periodic signal

# psd
Build a SIMPUT by defining a power spectral density
